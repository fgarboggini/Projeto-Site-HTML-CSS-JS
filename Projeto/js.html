Tag <a>

A tag <a> é usada para criar hyperlinks. O atributo href especifica o destino do link.

<a href="https://www.example.com">Visite Example.com</a>

Tag <p>

A tag <p> define um parágrafo.

<p>Este é um parágrafo.</p>
